To run the code you need to 

1: Have "sharp" and "node" downloaded.

2: Type "node favicon.js" in the terminal.

3: In the terminal you need to copy the URL of the site and paste it into any browser.

4: Using the URL, change the specified size to the one you need.

127.0.0.1:3000 - server

/img/100/100 - picture 

The first number is the width, then the height.
